# 🚀 Инструкция по запуску проекта e-qagaz

Эта инструкция содержит все необходимые шаги для запуска проекта на вашем компьютере и вывода его в интернет.

---

## 1. Локальный запуск (Разработка)

Проект состоит из двух частей: бэкенда (серверная логика на NestJS) и фронтенда (пользовательский интерфейс на NextJS). Для работы приложения должны быть запущены **обе части**.

### Шаг 1: Запуск Бэкенда
1. Откройте новый терминал.
2. Перейдите в папку бэкенда:
   ```bash
   cd backend
   ```
3. Запустите сервер:
   ```bash
   npm run start:dev
   ```
*(Бэкенд запустится на порту 3001)*

### Шаг 2: Запуск Фронтенда
1. Откройте **второй** (новый) терминал, оставив первый работать.
2. Перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```
3. Запустите приложение:
   ```bash
   npm run dev
   ```
*(Фронтенд запустится на порту 3000. Локальный адрес сайта: http://localhost:3000)*

---

## 2. Как вывести проект в Интернет (Доступ со смартфона и других ПК)

Чтобы любой человек мог зайти на ваш сайт, пока ваш компьютер включен, используйте встроенную функцию SSH-туннеля. Это безопасно, бесплатно и не требует установок.

1. Убедитесь, что фронтенд и бэкенд уже запущены (см. пункт 1).
2. Откройте **третий** терминал.
3. Скопируйте и вставьте эту команду:
   ```bash
   ssh -p 443 -R0:localhost:3000 a.pinggy.io
   ```
   *(Если Windows спросит «Are you sure you want to continue connecting?», напечатайте слово `yes` и нажмите Enter).*
4. В терминале появится табличка, внутри которой будет публичная ссылка вида `https://xxxxxx.a.pinggy.link`.
5. Отправьте эту ссылку друзьям или откройте на телефоне — сайт будет полностью работать!

## 3. Запуск 24/7 (Для домашнего сервера)

Если вы хотите, чтобы сайт работал постоянно и сам перезапускался при перезагрузке компьютера или сбоях, мы используем менеджер процессов **PM2**.

### Установка и запуск:
1. Откройте терминал в корневой папке `e-qagaz`.
2. Установите PM2 глобально:
   ```bash
   npm install -g pm2
   ```
3. Запустите Бэкенд через PM2:
   ```bash
   cd backend
   pm2 start npm --name "backend" -- run start:dev
   ```
4. Запустите Фронтенд через PM2:
   ```bash
   cd ../frontend
   pm2 start npm --name "frontend" -- run dev
   ```
5. Сохраните список процессов, чтобы они восстанавливались после перезагрузки:
   ```bash
   pm2 save
   ```

### Полезные команды PM2:
- `pm2 status` — посмотреть список работающих процессов и их статус.
- `pm2 logs` — посмотреть консоль (ошибки и логи) всех запущенных процессов.
- `pm2 stop all` — остановить все процессы.
- `pm2 restart all` — перезапустить все процессы.
